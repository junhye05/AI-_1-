## 2. ChatGPT의 답변

```
재밌는 질문이다 😄 센스 있게 답해볼게!

1️⃣ 내가 생각하는 가장 뛰어난 AI (나 제외)

솔직히 말하면 요즘 AI들은 각자 잘하는 분야가 달라서 “절대 1등”은 없어.
그래도 인상적인 AI 몇 개를 꼽아보면 이 정도야.

Claude
→ 긴 글 이해, 논리적인 글쓰기, 분석 능력이 뛰어남. 연구·문서 작업에 강함.

Gemini
→ Google이 만든 AI라서 검색 능력과 멀티모달(이미지·영상 이해)이 강함.

GitHub Copilot
→ 개발자들 사이에서는 거의 “자동완성 신” 수준. 코드 생산성이 매우 높음.

정리하면

글쓰기 / 분석 → Claude
검색 / 멀티모달 → Gemini
코딩 → Copilot

AI 세계는 약간 어벤져스 팀 같은 느낌이야. 각자 능력치가 다름 😎


2️⃣ AI에게 질문을 잘하는 방법

AI는 질문 수준 = 답변 퀄리티라고 생각하면 돼.

좋은 질문 공식 하나 알려줄게.

“R.C.C.P 공식”

R (Role) → 역할 지정  
C (Context) → 상황 설명  
C (Condition) → 조건  
P (Product) → 원하는 결과 형태

예시 비교

나쁜 질문
SQL Injection 설명해줘

좋은 질문
너는 정보보안 강사라고 생각하고
정보보안 전공 2학년 학생이 이해할 수 있게
SQL Injection을 예시 + 공격 원리 + 방어 방법으로 설명해줘

질문 방식만 바꿔도 답변 퀄리티가 훨씬 좋아짐.

추가 팁

- 대상 수준 말하기 (고등학생 수준, 전공자 기준 등)
- 결과 형식 요청 (표, 단계별 설명, 시험 대비 요약)
- 목적 말하기 (발표용, 보고서용 등)

AI는 결국 “질문을 잘하는 사람에게 더 똑똑해지는 도구”야.


3️⃣ 정보보안 컨설턴트를 위해 하면 좋은 활동

보안 업계에서 실제로 도움이 되는 활동 4가지를 소개할게.

1. CTF 대회 참여
해킹 문제를 해결하는 보안 대회로 실력 향상에 매우 도움이 됨.

2. 취약점 분석 플랫폼 연습
Hack The Box
TryHackMe

실제 해킹 환경을 체험하면서 웹·네트워크 취약점을 공부할 수 있음.

3. 기술 블로그 운영
보안 공부 내용을 정리하고 기록하면 성장 속도가 빨라짐.

예시 글
- SQL Injection 실습 정리
- Linux 권한 상승 정리
- 웹 해킹 공부 로드맵

4. 보안 커뮤니티 활동
OWASP 프로젝트 참여
보안 세미나 참석
해킹 동아리 활동

보안 업계는 커뮤니티 네트워크도 중요한 편이야.


보안 컨설턴트를 준비하는 현실적인 성장 순서

1. Linux + 네트워크 기초
2. 웹 해킹 이해
3. CTF 경험
4. 취약점 분석 실습
5. 기술 블로그 정리

이 루트 타면 보안 컨설턴트 / 모의해킹 직무에 가까워짐.
```

# Activity4 - 자유탐색 및 정리

## 질문

## ChatGPT의 답변

```id="s9d2k3"
처음 사용하는 사람이 쉽게 이해할 수 있도록 ChatGPT의 주요 기능을 간단하고 재미있게 정리해볼게 😊
ChatGPT는 단순한 “질문 답변 AI”가 아니라 여러 역할을 할 수 있는 AI 도구라고 생각하면 좋아.

1. 질문하고 답을 얻는 기능

가장 기본적인 기능이다.
궁금한 것을 물어보면 설명, 정리, 예시까지 함께 알려준다.

예시
“정보보안이 뭐야?”
“SQL Injection 쉽게 설명해줘”
“대학교 발표 주제 추천해줘”

→ 마치 24시간 질문 가능한 개인 튜터 같은 느낌이다.


2. 공부 도우미 기능

학생들이 가장 많이 사용하는 기능이다.

할 수 있는 것
- 어려운 개념 쉽게 설명
- 강의 내용 요약
- 시험 대비 핵심 정리
- 문제 풀이 과정 설명

예시
“네트워크 3계층 구조 쉽게 설명해줘”
“이 글 시험 대비용으로 요약해줘”

→ AI 공부 파트너라고 볼 수 있다.


3. 글쓰기 & 창작 기능

글을 작성하거나 아이디어를 만드는 데 도움을 줄 수 있다.

가능한 것
- 발표문 작성
- 보고서 초안 작성
- 소설 창작
- 자기소개서 아이디어

예시
“2030년 대학생활을 주제로 짧은 소설 써줘”
“발표용 도입 문장 만들어줘”

→ 작가 + 아이디어 메이커 역할을 할 수 있다.


4. 아이디어 브레인스토밍

아이디어가 필요할 때 도움을 줄 수 있다.

예시
- 프로젝트 주제 추천
- 창업 아이디어
- 동아리 활동 아이디어

예시 질문
“정보보안 관련 팀 프로젝트 아이디어 5개 알려줘”

→ 아이디어 회의 파트너 같은 역할이다.


5. 코딩 & 기술 도움

개발이나 IT 공부에도 많이 사용된다.

가능한 것
- 코드 설명
- 오류 해결
- 프로그램 예시 작성

예시
“Python으로 간단한 로그인 코드 만들어줘”
“이 코드 오류 이유 설명해줘”

→ 개발자에게는 AI 코딩 도우미가 되는 기능이다.


6. 번역 & 언어 도움

여러 언어를 번역하거나 문장을 자연스럽게 고쳐줄 수 있다.

예시
- 영어 → 한국어 번역
- 문장 자연스럽게 수정
- 영어 이메일 작성

예시 질문
“이 문장을 자연스러운 영어로 바꿔줘”

→ AI 번역가 + 문장 교정가 역할이다.


7. 정리 & 구조화

복잡한 정보를 표나 단계로 정리해줄 수 있다.

가능한 것
- 비교표 만들기
- 단계별 설명
- 체크리스트 작성

예시
“정보보안 직무 종류 표로 정리해줘”

→ 정리 전문가 같은 기능이다.


정리하면 ChatGPT는 다음과 같은 역할을 할 수 있다.

- 질문 답변 AI
- 공부 도우미
- 글쓰기 도우미
- 아이디어 브레인스토밍
- 코딩 도우미
- 번역 도우미
- 정리 도우미

즉, “대화로 사용하는 멀티 도구 AI”라고 볼 수 있다.
```

## 느낀점

오늘 실습에서는 ChatGPT를 활용해 질문을 하고 답변을 받아보는 활동을 진행하였다.
나는 AI에게 세 가지 질문을 하고 그 답변을 공유하며 AI가 어떤 방식으로 정보를 제공하는지 살펴보았다. 또한 AI와 함께 짧은 이야기를 만들어 보면서 창작 활동에도 활용할 수 있다는 점을 경험했다.

이번 실습을 통해 AI가 단순히 질문에 답하는 프로그램이 아니라 학습, 아이디어 생성, 글쓰기 등 다양한 분야에서 활용될 수 있다는 것을 알게 되었다. 특히 질문을 구체적으로 할수록 더 좋은 답변을 얻을 수 있다는 점이 인상적이었다.

또한 AI와 함께 이야기를 만드는 과정이 생각보다 재미있었고, 발표 자료를 준비할 때도 도움이 될 수 있을 것이라고 느꼈다.

앞으로는 공부를 하면서 이해하기 어려운 개념을 질문하거나 과제를 준비할 때 아이디어를 얻는 등 AI를 학습 도구로 적극 활용해 보고 싶다. 이번 실습을 통해 AI를 올바르게 활용하면 학습에 큰 도움이 될 수 있다는 것을 느낄 수 있었다.
